## Сниппет 2 — is_prime

```cpp
bool is_prime(int n) {
    for (int i = 2; i < n; i++) {
        if (n % i == 0) {
            return false;
        }
    return true; // ❌ закрывающая скобка цикла пропущена
}
```

**Ошибки:**
1. Отсутствует `}` закрывающий тело `for` — `return true` внутри цикла, вернёт `true` уже на первой итерации
2. `is_prime(1)` вернёт `true`, хотя 1 не простое — нужна проверка `if (n < 2) return false`

**Исправление:**
```cpp
bool is_prime(int n) {
    if (n < 2) return false;
    for (int i = 2; i < n; i++) {
        if (n % i == 0) return false;
    }
    return true;
}
```

## Сниппет 3 — reverse_string

```cpp
void reverse_string(std::string s) { // ❌ передача по значению
    int n = s.length();
    for (int i = 0; i < n; i++) {    // ❌ проходит весь массив — разворачивает дважды
        std::swap(s[i], s[n - i - 1]);
    }
    std::cout << s << std::endl;
}
int main() {
    reverse_string(text);
    std::cout << text << std::endl; // выведет оригинал — копия не вернулась
}
```

**Ошибки:**
1. Цикл `i < n` разворачивает строку дважды — в итоге она остаётся исходной. Нужно `i < n / 2`
2. Строка передаётся по значению — изменения не видны снаружи

**Исправление:**
```cpp
void reverse_string(std::string& s) {
    int n = s.length();
    for (int i = 0; i < n / 2; i++) {
        std::swap(s[i], s[n - i - 1]);
    }
}
```

## Сниппет 5 — count_uppercase

```cpp
for (int i = 0; i < s.size(); i++) {
    if (s[i] >= 65 && s[i] <= 90) { // ❌ магические числа
```

**Ошибки:**
1. Магические числа вместо читаемых символов — лучше `s[i] >= 'A' && s[i] <= 'Z'`
2. Можно использовать `std::isupper` из `<cctype>`

**Исправление:**
```cpp
int count_uppercase(const std::string& s) {
    int count = 0;
    for (char c : s)
        if (std::isupper(c)) count++;
    return count;
}
```

## Сниппет 6 — file reading

```cpp
while (!file.eof()) {           // ❌ антипаттерн
    std::getline(file, line);
    std::cout << line << std::endl;
}
```

**Ошибки:**
1. `while (!file.eof())` — классический баг: последняя строка выводится дважды, так как `eof` устанавливается только после неудачного чтения

**Исправление:**
```cpp
while (std::getline(file, line)) {
    std::cout << line << std::endl;
}
```
