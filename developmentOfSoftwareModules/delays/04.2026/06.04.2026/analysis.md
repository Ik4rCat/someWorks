# Анализ классов и методов Qt — 06.04.2026

## 1. MyWidget

**Назначение:** виджет с кастомной отрисовкой фигур и текста.  
**Наследует:** `QWidget`

### Методы

---

#### `paintEvent(QPaintEvent *event)` — protected, override
**Принимает:** указатель на `QPaintEvent` (содержит информацию об области, требующей перерисовки).  
**Что делает:** вызывается Qt автоматически при необходимости перерисовать виджет. Внутри:

| Вызов | Что делает |
|---|---|
| `QPainter painter(this)` | Создаёт контекст рисования на данном виджете |
| `painter.setRenderHint(QPainter::Antialiasing)` | Включает сглаживание краёв фигур |
| `painter.setPen(Qt::red)` | Устанавливает цвет контура — красный |
| `painter.setBrush(Qt::blue)` | Устанавливает цвет заливки — синий |
| `painter.drawRect(50, 50, 100, 100)` | Рисует прямоугольник: x=50, y=50, ширина=100, высота=100 |
| `painter.setPen(Qt::green)` | Меняет цвет контура на зелёный |
| `painter.drawEllipse(200, 50, 100, 100)` | Рисует эллипс в прямоугольнике x=200, y=50, 100×100 |
| `painter.drawText(50, 200, "Hello Qt!")` | Выводит строку в точке (50, 200) |
| `QPen pen(Qt::red, 2, Qt::DashLine)` | Создаёт перо: красный, толщина 2, пунктир |
| `painter.drawLine(50, 250, 300, 250)` | Рисует линию из (50,250) в (300,250) |

---

## 2. TimerWidget

**Назначение:** виджет с анимацией движущегося круга и счётчиком секунд через `QTimer`.  
**Наследует:** `QWidget`  
**Макрос:** `Q_OBJECT` — обязателен для сигналов и слотов

### Поля

| Поле | Тип | Назначение |
|---|---|---|
| `timer` | `QTimer*` | Периодический таймер |
| `counter` | `int` | Счётчик прошедших секунд |
| `elapsedTime` | `QTime` | Объект времени (объявлен, не используется в коде) |

### Методы

---

#### `TimerWidget(QWidget *parent)` — конструктор
**Принимает:** `parent` — родительский виджет (по умолчанию `nullptr`).  
**Что делает:**
- Инициализирует `counter = 0`
- Создаёт `QTimer`
- `QTimer::singleShot(3000, this, &TimerWidget::oneShotSlot)` — запускает одноразовый таймер на 3000 мс (3 секунды), после чего вызовет `oneShotSlot`
- Подключает сигнал `timeout` таймера к слоту `updateTimer`
- `timer->start(1000)` — запускает периодический таймер с интервалом 1000 мс

---

#### `updateTimer()` — private slot
**Принимает:** ничего.  
**Что делает:** вызывается каждую секунду. Увеличивает `counter` на 1, вызывает `update()` для перерисовки виджета.

---

#### `oneShotSlot()` — private slot
**Принимает:** ничего.  
**Что делает:** вызывается один раз через 3 секунды после запуска. Выводит отладочное сообщение через `qDebug()`.

---

#### `paintEvent(QPaintEvent *event)` — protected, override
**Принимает:** указатель на `QPaintEvent`.  
**Что делает:**
- Выводит текст с количеством прошедших секунд: `"Прошло секунд: N"`
- Вычисляет позицию X движущегося круга: `(counter * 5) % width()` — круг движется вправо, при достижении края появляется слева
- Рисует красный эллипс диаметром 30×30 в вычисленной позиции

---

## 3. CustomWidget

**Назначение:** виджет с анимацией вращающегося прямоугольника через низкоуровневый `timerEvent`.  
**Наследует:** `QWidget`

### Поля

| Поле | Тип | Назначение |
|---|---|---|
| `timerId` | `int` | Идентификатор таймера, возвращённый `startTimer()` |
| `animationValue` | `int` | Текущий угол вращения (0–359°) |

### Методы

---

#### `CustomWidget(QWidget *parent)` — конструктор
**Принимает:** `parent` — родительский виджет.  
**Что делает:** инициализирует `animationValue = 0`, запускает таймер с интервалом 50 мс через `startTimer(50)`, сохраняет его ID в `timerId`.

---

#### `timerEvent(QTimerEvent *event)` — protected, override
**Принимает:** указатель на `QTimerEvent` (содержит ID сработавшего таймера).  
**Что делает:** проверяет, что сработал именно наш таймер (`event->timerId() == timerId`). Увеличивает `animationValue` на 5, выполняет обёртку по модулю 360 (полный круг), вызывает `update()`.

---

#### `paintEvent(QPaintEvent *event)` — protected, override
**Принимает:** указатель на `QPaintEvent`.  
**Что делает:**
- `painter.translate(width()/2, height()/2)` — переносит систему координат в центр виджета
- `painter.rotate(animationValue)` — поворачивает систему координат на текущий угол
- `painter.drawRect(-50, -25, 100, 50)` — рисует прямоугольник 100×50 с центром в начале координат (т.е. в центре виджета)

---

## 4. AnimationWidget

**Назначение:** виджет с анимацией отскакивающего мячика, меняющего цвет при отскоке.  
**Наследует:** `QWidget`  
**Макрос:** `Q_OBJECT`  
**Q_PROPERTY:** `radius` — свойство доступно через систему мета-объектов Qt (геттер `radius()`, сеттер `setRadius()`)

### Поля

| Поле | Тип | Назначение |
|---|---|---|
| `animationTimer` | `QTimer*` | Таймер анимации (~50 FPS) |
| `radius` | `int` | Радиус мячика |
| `x, y` | `int` | Текущая позиция центра мячика |
| `dx, dy` | `int` | Скорость по X и Y (пикселей за кадр) |
| `color` | `QColor` | Текущий цвет мячика |

### Методы

---

#### `AnimationWidget(QWidget *parent)` — конструктор
**Принимает:** `parent` — родительский виджет.  
**Что делает:** инициализирует все поля (radius=30, x=50, y=50, dx=3, dy=2, color=синий), создаёт таймер, подключает его сигнал `timeout` к слоту `animate`, запускает таймер с интервалом 20 мс (~50 FPS), устанавливает минимальный размер виджета 400×300.

---

#### `radius()` — public, const
**Принимает:** ничего.  
**Возвращает:** `int` — текущий радиус мячика. Геттер для Q_PROPERTY.

---

#### `setRadius(int r)` — public
**Принимает:** `int r` — новый радиус.  
**Что делает:** присваивает `radius = r`, вызывает `update()` для немедленной перерисовки. Сеттер для Q_PROPERTY.

---

#### `animate()` — private slot
**Принимает:** ничего.  
**Что делает:** основная логика анимации за один кадр:
- Обновляет позицию: `x += dx`, `y += dy`
- Проверяет выход за правую/левую границу: `x + radius >= width()` или `x - radius <= 0` → инвертирует `dx`, вызывает `changeColor()`
- Проверяет выход за верхнюю/нижнюю границу: `y + radius >= height()` или `y - radius <= 0` → инвертирует `dy`, вызывает `changeColor()`
- Вызывает `update()` для перерисовки

---

#### `changeColor()` — private slot
**Принимает:** ничего.  
**Что делает:** меняет цвет мячика, сдвигая оттенок (Hue) в HSV-пространстве на 30°: `QColor::fromHsv((color.hue() + 30) % 360, 255, 255)`. Это обеспечивает плавную смену цветов по кругу при каждом отскоке.

---

#### `paintEvent(QPaintEvent *event)` — protected, override
**Принимает:** указатель на `QPaintEvent`.  
**Что делает:**
- Включает сглаживание
- `painter.fillRect(rect(), Qt::white)` — заливает весь виджет белым
- Рисует мячик: `drawEllipse(QPoint(x, y), radius, radius)` — эллипс с центром в (x,y) и радиусами radius×radius
- Выводит текст с текущей позицией: `"Position: (x, y)"`

---

#### `resizeEvent(QResizeEvent *event)` — protected, override
**Принимает:** указатель на `QResizeEvent` (содержит старый и новый размер).  
**Что делает:** при изменении размера окна корректирует позицию мячика через `qBound(radius, x, width() - radius)` и аналогично для Y — не даёт мячику выйти за новые границы. Вызывает базовую реализацию `QWidget::resizeEvent(event)`.
