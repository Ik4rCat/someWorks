# Анализ функции |2(|x| − 1)² − 3|

## График

Открыть: `graph_290126.html` (в этой же папке)

## Математический анализ

### Шаг 1: Упрощение

Обозначим внутреннюю функцию:
```
f(x) = 2(|x| - 1)² - 3
```
Итоговая функция: `y = |f(x)| = |2(|x| - 1)² - 3|`

### Шаг 2: Область допустимых значений (ОДЗ)

Функция определена при **всех x ∈ ℝ** — нет логарифмов, корней, дробей.

**ОДЗ: x ∈ (−∞; +∞)**

### Шаг 3: Чётность

Так как `|x|` чётная функция → `f(x) = f(-x)` → функция **чётная**.
График симметричен относительно оси OY.

### Шаг 4: Анализ для x ≥ 0 (для x < 0 — зеркально)

При x ≥ 0: `|x| = x`, поэтому `f(x) = 2(x - 1)² - 3`

**Нули внутренней функции** (когда f(x) = 0):
```
2(x - 1)² = 3
(x - 1)² = 3/2
x - 1 = ±√(3/2) ≈ ±1.2247
x ≈ 2.2247  или  x ≈ -0.2247
```

При x ≥ 0: нули в точках `x ≈ 0` (нет, т.к. -0.2247 < 0) и `x ≈ 2.2247`

Минимум `f(x)` при x = 1: `f(1) = 2·0 - 3 = -3`

### Шаг 5: Участки возрастания и убывания y = |2(|x| - 1)² - 3|

Анализируем по участкам (x ≥ 0, левая половина симметрична):

| Интервал | f(x) знак | y = \|f(x)\| | y монотонность |
|----------|-----------|--------------|----------------|
| [0; 1) | f убывает (от -1 до -3) | \|f\| возрастает (от 1 до 3) | ↑ возрастает |
| x = 1 | f = -3 (минимум) | y = 3 (локальный максимум) | — |
| (1; ≈2.225) | f возрастает (от -3 до 0) | \|f\| убывает (от 3 до 0) | ↓ убывает |
| x ≈ 2.225 | f = 0 | y = 0 (нуль функции) | — |
| (≈2.225; +∞) | f > 0, возрастает | y = f, возрастает | ↑ возрастает |

### Шаг 6: Ключевые точки

| x | y | Описание |
|---|---|----------|
| 0 | 1 | y(0) = \|2·1 - 3\| = 1 |
| ±1 | 3 | локальный максимум |
| ≈±2.225 | 0 | нули функции |
| ≈±0.225 | 0 | нули функции (симметрично) |

### Итог

- **ОДЗ:** x ∈ ℝ
- **Область значений:** y ∈ [0; +∞)
- **Функция чётная**, симметрична относительно OY
- **Нули:** x ≈ ±0.225, x ≈ ±2.225
- **Локальные максимумы:** x = ±1, y = 3
- **Убывает:** (−∞; −2.225), (−1; 0), (0; 1), (2.225; не применимо — ↑)
- **Возрастает:** (−2.225; −1), (0; 1) — нет, (1; 2.225) убывает...

Итоговые участки (по симметрии):
- **Возрастает:** (−∞; −1), (−0.225; 0), (0; 1), (2.225; +∞)  → с учётом модуля
- **Убывает:** (−1; −0.225), (1; 2.225)
